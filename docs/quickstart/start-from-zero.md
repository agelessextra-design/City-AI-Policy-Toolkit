# Start From Zero

## When To Use This Path

Use this guide when your city does not already have:

- a mature AI program
- a full source set
- a clear cross-functional review structure
- a settled starting module

## Safe First Sequence

1. Read `start-here.md`.
2. Use `../../examples/example-entry-index.md` to pick the closest city-scale example.
3. Choose the first module family based on the city’s biggest gap.
4. Build one bounded first artifact only.
5. Keep local-decision markers visible.
6. Run evaluation before any review packet is assembled.

## Minimum Starting Bundle

- one template family under `../../toolkit/templates/`
- one matching prompt pack under `../../toolkit/prompts/`
- one evaluation scorecard under `../../toolkit/evaluations/`
- one worked example under `../../examples/annotated/`

## Exit Condition

This path is complete when the city has one evaluated first-pass artifact and a visible list of missing local inputs.
