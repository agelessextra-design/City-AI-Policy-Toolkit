# Recover Missing Inputs

## Purpose

Use this guide when your city lacks enough local information to complete a clean first draft.

The correct move is to expose the gap, not hide it.

## Recovery Rules

- convert the missing input into a visible placeholder
- narrow the artifact scope before expanding it
- use examples for structure only
- do not move to review before the missing input is either supplied or explicitly logged

## Typical Recovery Paths

- if strategic direction is missing, start with strategic leadership artifacts
- if approval logic is missing, start with governance artifacts
- if operational detail is missing, start with operations artifacts
- if resident-facing accountability is missing, start with community-trust artifacts
