# Quickstart

This folder should contain the shortest safe starting path for a city team using the toolkit.

The public quickstart should answer:

1. which module family to start with
2. what local inputs are needed before drafting
3. why outlines come before prose
4. when evaluation and review happen
5. what materials are examples versus reusable toolkit assets

Minimum quickstart topics:

- AI-assistant use path
- strategic leadership start path
- governance and policy start path
- workforce and learning start path
- operations and service delivery start path
- community trust start path
- start-from-zero path
- missing-input recovery path

Do not copy internal workspace orchestration notes here. Rewrite for standalone public use.

Start with `use-with-ai-assistants.md` when a city wants Codex, Claude Code, or another assistant to help use the toolkit.
