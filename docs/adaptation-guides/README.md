# Adaptation Guides

This folder should contain reusable guidance for adapting toolkit assets to a city without hiding local choices.

Public adaptation guides should help users:

- preserve placeholders and local-decision markers
- decide when research is required instead of guessing
- map local roles and authorities safely
- adapt examples without turning them into default policy text
- distinguish reusable structure from city-specific commitments

Place only reusable guidance here that helps cities adapt toolkit assets without hiding local decisions or legal review needs.
