# Documentation

This folder contains workflow and orientation documentation for the public toolkit bundle.

Use it for the public-facing path into the toolkit:

- quickstart guidance
- AI-assistant usage guidance
- workflow overview
- adaptation philosophy and supporting documentation

The public docs should help a user:

- understand the five module families
- choose where to start
- use AI assistants without losing local decision control
- follow the intake -> outline -> draft -> evaluation -> review path
- distinguish reusable toolkit assets from examples

Do not place workspace-only planning notes here.

Only promote docs into this folder when they can stand on their own without requiring access to the internal build workspace.
