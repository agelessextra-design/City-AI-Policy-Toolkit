# Module Family Scorecard Template

## Purpose

Use this scorecard to evaluate one module family or one bounded artifact batch using the canonical six-dimension orchestration model.

This template is intended for section-level build loops and milestone checks, not for sentence-level redlining.

- strategic leadership artifacts
- governance and policy artifacts
- workforce and learning artifacts
- operations and service delivery artifacts
