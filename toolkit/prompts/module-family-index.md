# Prompt Module Family Index

## Purpose

This index tells a public user which prompt family to start with and when to move from build packs to hardening packs.

## Shared Pattern

Each module family should support:

- intake
- outline
- draft
- evaluation
- hardening

Hardening includes:

- repair
- contradiction resolution
- score-lift or upgrade work
- board-prep work

## Selection Rule

- use the build pack first
- move to the hardening pack after the first evaluation pass or when contradictions appear
- keep the prompt family aligned with the template family you selected
