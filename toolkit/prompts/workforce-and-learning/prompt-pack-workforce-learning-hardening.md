# Prompt Pack: Workforce Learning Hardening

## Purpose

This prompt pack is used after the first workforce build pass to repair issues, resolve contradictions, lift low scores, and prepare workforce artifacts for board-style review.

## Shared Inputs

- `toolkit/prompts/workforce-and-learning/prompt-pack-workforce-learning.md`
- `toolkit/evaluations/guide-module-family-review-readiness.md`
- `toolkit/evaluations/template-module-family-scorecard.md`
- `toolkit/evaluations/template-shared-issue-log.md`
- `toolkit/review-sprint-kit/template-contradiction-sweep-checklist.md`
- relevant workforce templates and adapter notes
- current issue log and contradiction sweep results

## Pack 5: Workforce Repair

Return:

1. issue-by-issue repair plan
2. exact sections to revise
3. unresolved labor, HR, or support decisions
4. revised handoff target

Rules:

- do not invent labor posture, staffing effects, or manager authority
- keep workforce implications tied to real use cases

## Pack 6: Workforce Contradiction Resolution

Use when workforce artifacts conflict with strategy, governance, operations, or trust assumptions.

Return:

1. contradiction inventory
2. preferred resolution path
3. impacted artifacts
4. required owner decisions

## Pack 7: Workforce Upgrade

Use when the workforce plan is below target but structurally usable.

Return:

1. score-lift plan by dimension
2. highest-leverage rewrites
3. missing evidence or companion artifacts
4. next evaluation checkpoint

## Pack 8: Workforce Board Prep

Return:

1. likely board objections
2. proof points already present
3. weak spots needing hardening
4. review packet additions needed before board simulation
