# Prompt Pack: Community Trust Hardening

## Purpose

This prompt pack is used after the first community-trust build pass to repair issues, resolve contradictions, lift low scores, and prepare trust artifacts for board-style review.

## Shared Inputs

- `toolkit/prompts/community-trust/prompt-pack-community-trust.md`
- `toolkit/evaluations/guide-module-family-review-readiness.md`
- `toolkit/evaluations/template-module-family-scorecard.md`
- `toolkit/evaluations/template-shared-issue-log.md`
- `toolkit/review-sprint-kit/template-contradiction-sweep-checklist.md`
- relevant community-trust templates and adapter notes
- current issue log and contradiction sweep results

## Pack 5: Community Trust Repair

Return:

1. issue-by-issue repair plan
2. exact sections to revise
3. unresolved notice, accessibility, engagement, or redress decisions
4. revised handoff target

Rules:

- do not imply disclosure alone solves trust gaps
- keep resident support and escalation paths visible

## Pack 6: Community Trust Contradiction Resolution

Use when trust artifacts conflict with governance, operations, or communications assumptions.

Return:

1. contradiction inventory
2. preferred resolution path
3. impacted artifacts
4. required owner decisions

## Pack 7: Community Trust Upgrade

Use when the trust framework or service notice is usable but below target.

Return:

1. score-lift plan by dimension
2. highest-leverage rewrites
3. missing evidence or companion artifacts
4. next evaluation checkpoint

## Pack 8: Community Trust Board Prep

Return:

1. likely board objections
2. proof points already present
3. weak spots needing hardening
4. review packet additions needed before board simulation
