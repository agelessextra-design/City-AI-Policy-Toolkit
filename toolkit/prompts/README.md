# Prompts

This folder should contain reusable prompt packs and prompt guidance.

Prompt packs should be organized so a standalone user can run:

- intake
- outline
- draft
- evaluation

Later hardening packs may include:

- repair
- contradiction resolution
- low-score-to-high-score upgrade
- board-review preparation

Only promote prompts that can stand without internal workspace coordination context.
