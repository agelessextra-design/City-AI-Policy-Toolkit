# Prompt Pack: Strategic Leadership Hardening

## Purpose

This prompt pack is used after the first strategic leadership build pass to repair weaknesses, resolve contradictions, lift low scores, and prepare the charter and portfolio review artifacts for board-style review.

## Shared Inputs

- `toolkit/prompts/strategic-leadership/prompt-pack-strategic-leadership.md`
- `toolkit/evaluations/guide-module-family-review-readiness.md`
- `toolkit/evaluations/template-module-family-scorecard.md`
- `toolkit/evaluations/template-shared-issue-log.md`
- `toolkit/review-sprint-kit/template-contradiction-sweep-checklist.md`
- relevant strategic leadership templates and adapter notes
- current issue log and contradiction sweep results

## Pack 5: Strategic Leadership Repair

Return:

1. issue-by-issue repair plan
2. exact sections to revise
3. unresolved leadership or budget decisions that block clean repair
4. revised handoff target

Rules:

- do not invent executive ownership or portfolio maturity
- keep cross-family dependencies explicit
- if ownership remains unresolved, add interim owner scope, blocked actions, and decision deadlines before treating the draft as stabilized

## Pack 6: Strategic Leadership Contradiction Resolution

Use when strategy language conflicts with governance, workforce, operations, or trust artifacts.

Return:

1. contradiction inventory
2. preferred resolution path
3. impacted artifacts
4. required leadership decisions

## Pack 7: Strategic Leadership Upgrade

Use when the charter or portfolio memo is below target but fundamentally sound.

Return:

1. score-lift plan by dimension
2. highest-leverage rewrites
3. missing evidence or secondary artifacts
4. next evaluation checkpoint

## Pack 8: Strategic Leadership Board Prep

Return:

1. likely board objections
2. proof points already present
3. weak spots needing hardening
4. review packet additions needed before board simulation
