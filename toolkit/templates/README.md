# Templates

This folder should contain reusable public templates plus companion adapter notes.

The public template layer should expose module-family entry points for:

- strategic leadership
- governance and policy
- workforce and learning
- operations and service delivery
- community trust

Template rules:

- keep placeholders visible
- keep `[[REQUIRES_LOCAL_DECISION: ...]]` markers visible until resolved
- keep `[[REQUIRES_LOCAL_RESEARCH: ...]]` and `[[REQUIRES_LEGAL_REVIEW: ...]]` markers visible where needed
- keep city-specific example text out of the reusable core
- pair major templates with adapter notes
