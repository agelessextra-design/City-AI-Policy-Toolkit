# Toolkit Modules

This folder contains the reusable toolkit modules for the public city AI policy toolkit.

Current packaging target:

- `templates/` for reusable module families and companion adapter notes
- `prompts/` for intake, outline, draft, evaluation, and later hardening prompt packs
- `evaluations/` for rubrics, issue logs, and release-gate evidence
- `review-sprint-kit/` for human review workflow assets
- `education/` for quickstart and adaptation guidance

The public toolkit should make it easy to find these five module families:

- strategic leadership
- governance and policy
- workforce and learning
- operations and service delivery
- community trust

Only place reusable, reviewable toolkit assets here. Keep provenance records, source-audit working papers, and internal planning material out of this folder.
