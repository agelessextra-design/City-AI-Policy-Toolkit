# Review Sprint Kit

This folder should contain reusable public review-sprint materials.

The public review-sprint layer should include:

- packet-assembly guidance
- reviewer role guidance
- issue and decision logging guidance
- bounded sprint mechanics
- escalation and rework routing guidance

Do not place internal decision logs or private review notes here unless they have been rewritten as reusable public assets.

Current promoted review-sprint entry assets:

- `review-entry-index.md`
- `template-review-sprint-brief.md`
- `template-review-sprint-routing-register.md`
- `guide-issue-aging-and-escalation.md`
- `template-board-review-packet.md`
- `template-unresolved-major-issue-escalation-brief.md`
