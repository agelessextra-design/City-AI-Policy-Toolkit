# Examples

This folder contains annotated examples for the public toolkit bundle.

Examples in this folder should:

- remain clearly labeled as examples
- preserve adaptation notes where local decisions matter
- stay distinct from reusable core toolkit material
- help users see how the five module families connect in practice

Do not treat example content as default city policy text.
